# HiyokoQuest  
---  
# Description  
個人で作成したローグライクゲームです.  
ローグライクの基本要素のみの単純なゲームシステムです.

# 遊び方  
1. 基本操作  
 WASDキー: 上左下右  
 Eキー: 攻撃  
2. Title画面  
 WSキー: 上下移動  
 Xキー: スタート  
3. Game画面  
 WASDキー: 上左下右(2入力可)  
 Eキー: 攻撃  
 F1キー: 速度切り替え(3段階)
4. Result画面  
 Sキー: Titleに戻る
